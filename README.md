# GoodNeighbors

### Good change for the world

글로벌 아동권리 전문 NGO 앱을 후원자들이 커뮤니티를 형성하여 활동하도록하여

후원 및 캠페인에 다양하게 참여할 수 있도록 만든 **후원자 친화적 앱**입니다.
